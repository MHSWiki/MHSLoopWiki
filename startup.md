# 🚀 How to Start the MHS Loop Wiki

Follow these steps to start the local development server and view the wiki in your browser. Whenever you want to work on the site or preview changes, you will need to do this.

## Step 1: Open Your Terminal
Open your terminal application (e.g., Terminal on Mac, or the integrated terminal inside your code editor like VS Code or Cursor).

## Step 2: Navigate to the Project Folder
Make sure you are in the correct folder for the project:
```bash
cd /Users/thewtd/Desktop/MHSWiki_claude
```

## Step 3: Start the Astro Server
Run the following command to start the local development server:
```bash
npm run dev
```

## Step 4: Open the Wiki in Your Browser
Once the server is running, the terminal will display a local network address. 
Open your preferred web browser (Chrome, Safari, etc.) and go to this exact URL:

👉 **http://localhost:4321/**

## 📝 How to Edit Content
1. Leave the terminal running in the background. Do not close it or hit `Ctrl+C` unless you want to stop the server.
2. Open any `.md` or `.mdx` file in the `src/content/docs/` folder using your code editor.
3. Make your text changes and save the file.
4. Your browser will instantly and automatically refresh to show the new content!
